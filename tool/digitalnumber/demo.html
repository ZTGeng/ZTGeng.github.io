Digital Numbers JS Demo

  • 此组件自动将数字转换为电子表数字样式。
  • 只需加载一个js文件及jquery（>3.5.1）。无需任何字体等依赖。
  • 转化后的数字样式为纯div实现，非字符，不支持拖选。
  • 进行转换的标签只能包含文字，不能包含下级标签（将被移除）。支持数字与非数字混排（只有数字会被转换）。
  • 支持对标签内容重新赋值，实时转换。
  • 浏览器兼容性要求：MutationObserver。

使用方法：

  1. 需加载jquery。
  2. 对需要转换数字的标签设置 class="digital-numbers-js"。
  3. 在文档末尾加载以下js文件:

    <script src="digital-numbers.js"></script>

效果：

静态文本：1234567890

    <div class="digital-numbers-js">静态文本：1234567890</div>

123

    <button id="random-number-button">点击刷新数字 -></button>
    <div class="digital-numbers-js" id="random-number">123</div>
    <script>
        $("#random-number-button").on("click", function() {
            $("#random-number").text(("" + Math.random()).slice(2, 5));
        });
    </script>

Time - 00:00:00

    <div class="digital-numbers-js" id="clock">Time - 00:00:00</div>
    <script>
        var twoDigits = function(num) {
            return (num < 10 ? "0" : "") + num;
        }
        setInterval(function() {
            var now = new Date();
            $("#clock").text("Time - " + twoDigits(now.getHours()) + ":" + twoDigits(now.getMinutes()) + ":" + twoDigits(now.getSeconds()));
        }, 1000);
    </script>

使用 data-height 与 data-color 自定义高度（px）与颜色：123

    <div>使用 <code>data-height</code> 与 <code>data-color</code> 自定义高度（px）与颜色：<span class="digital-numbers-js" data-height="50" data-color="red">123</span></div>
